Overview

PacificLinguistsClub.com is an appointment booking system that connects English language students with native English speakers to improve their speaking and listening skills.

Tools Used

AWS: Route 53, RDS, SSL, Load Balancing

Back-end: PHP, MySQL Database, PayPal API, MD5 Encryption

Front-end: HTML, CSS, JavaScript

Major Features

MySQL Database

PHP works with the MySQL database to execute prepared statements for creating an account, querying appointments, booking appointments, updating account settings, etc.

Booking System

A search tool helps users find appointments throughout the day/week.

PayPal Payments

Integration with the PayPal API allows payments to be received.

Japanese Translation

USA & Japan buttons appears at the top of each webpage to translate the page between English and Japanese.

Scheduler Tool

The user's schedule is made very easy to read. Users can easily create, update, and delete appointments by clicking on empty or existing appointment timeslots.

Other Features

The website contains about 30 English webpages (and another 30 in Japanese) for other features like updating user settings, tutor registration, confirming booked appointments, custom error pages, etc.

UX Design

Sketch Designs

Sketch was used to create mockups of the web app to help gather feedback from team members, identify usability issues, and aid development.

Color

The website uses green and blue colors to emphasize important clickable elements. White background and black text aids the visibility of these elements and improves readability. These colors also enforce the branding and company logo.

Japanese Audience

The translator tool exists at the top of each webpage to provide English and Japanese speakers a way to easily translate entire webpages.
